La Michel Documentation

Bienvenue

L'arcade est structuré sur un modele MVC (modele, vue, controleur), dans notre cas, le core va s'occupper de gerer dans les librairies graphiques dans le dossier "lib" et les jeux contenus dans le dossier "games". Le core va faire le lien entre le jeu et la librairie graphique permettant à celles-ci de communiquer.

Core

Changement de jeu: 
void switchGame(arcade::CommandType);



Cette fonction sert à changer de jeu, elle prend en paramètre un arcade::CommandType .
Le CommandType pris en paramètre sert à spécifier si on change de jeu vers le prochain ou vers le précédent de la list des jeux disponibles

Changement de lib: 
void switchLib(arcade::CommandType);



Cette fonction sert à changer de lib graphique, elle prend en paramètre un arcade::CommandType .
Le CommandType pris en paramètre sert à spécifier si on change de lib graphique vers la prochaine ou vers le précédent de la list des libs disponibles

Rafraichir le GUI: 
void refreshGui();



Cette fonction sert à rafraichir la GUI ( Affichage des libs graphique disponibles, Affichage du score ect... ).

Ecran de victoire: 
void gameWin();



Cette fonction sert à afficher l'écran de victoire.

Ecran de défaite: 
void gameOver();



Cette fonction sert à afficher l'écran de défaite.

Récupérer les Saves: 
Save &getSave();



Cette fonction sert à récuperer une Class de type Save.

Récupérer les sons: 
std::vector<std::string> getSounds();



Cette fonction sert à récuperer la liste des paths des sons a charger.

Récupérer la lib graphique: 
arcade::IGraph           *getLib() const;



Cette fonction sert à récuperer la lib graphique en cours d'utilisation.

Récupérer le jeu actuel: 
const std::string        &getCurrentGame() const;



Cette fonction sert à récuperer le nom de du jeu en cours d'utilisation.

Sauvegarder le score: 
void    saveScore(size_t score);



Cette fonction sert à sauvegarder le score sur le jeu en cours d'utilisation.

Set le score: 
void    setScore(const std::string &score);



Cette fonction sert à envoyer le score du jeu actuelement utilisé dans la GUI.

Games

Initialiser le jeu: 
IGame &createGame();



Crée une instance du jeu actuellement chargé par le Core.

Detruire le jeu: 
void deleteGame(IGame &);



Détruit le jeu actuellement chargé par le Core.

Lancer le jeu: 
arcade::CommandType  play(arcade::ICore &);



Lance le jeu chargé par le Core.

Graphique

Créer une fenêtre: 
void aInit(arcade::ICore *core, size_t HEIGHT, size_t WIDTH);



Ouvre une fenêtre, de HEIGHT par WIDTH.

Ferme la fenêtre: 
void aClose();



Ferme la fenêtre si celle ci est ouverte.

Afficher un cube dans la fenêtre: 
void aTile(size_t, size_t, int, arcade::TileType, arcade::CommandType);



Affiche un cube à une position x, y, puis on lui envoie la vitesse du block, le type du block à afficher et la direction du block pour l'animer.

Assigner une texture à un cube: 
void aAssignTexture(arcade::TileType tile, const std::string &path, arcade::Color color);



Cette fonction assigne une texture à un block de type TileType donnée en paramètre, grâce au PATH et une couleur par default en cas d'erreur de chargement de la texture. Cliquez ici pour voir l'enum Color

Lire un son: 
void aPlaySound(arcade::Sound);



Cette fonction lit par apport à l'action de type Sound donné en paramètre. Cliquez ici pour voir l'enum Sound

Lire une music: 
void aPlayMusic(const std::string &);



Cette fonction sert à lire une music par apport au PATH de celle-ci donné en paramètre ce cette fonction.

Afficher du texte: 
void aPutText(size_t, size_t, arcade::Font, size_t, arcade::Color, const std::string &);



Cette fonction sert à afficher du text à une position x, y, un font, la taille du texte, une couleur et le texte donné en paramètre.
Cliquez ici pour voir l'enum Font
Cliquez ici pour voir l'enum Color

Clear la fenêtre: 
void aClear();



Cette fonction sert à supprimer tout les block, texte etc... de la fenêtre.

Clear les animations: 
void aClearAnimBuffer();



Cette fonction sert à supprimer toute les annimations en cours d'utilisation.

Rafraichir la fenêtre: 
void aRefresh();



Cette fonction sert à afficher tout les block, textes etc.. dans la fenêtre.

Récupérer les commandes: 
arcade::CommandType aCommand();



Cette fonction sert à récuperer les commandes de type CommandType

Récupérer les touches: 
std::string         aChar();



Cette fonction sert à récuperer les touches de type texte (a, b, c, d, etc ...)
